### Default Arguments

```python

```

```python

```

```python

```

### Keyword Arguments

```python
def greet(msg, name):
    print(msg + " " + name)
```

```python
greet("Good morning", "Peter")
```

```python
greet(name="Peter", msg="Good night")
```

### Arbitrary Arguments

```python
def greet(*names):
    for name in names:
        print("Hello " + name)
```

```python

```
